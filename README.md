## 📌 프로젝트 시작하기

### 💕 Clone
 - `master` branch clone 부탁드립니다 :)
 
```js
git clone https://github.com/boostcamp-2020/Project16-B-Account-Book.git
```

### 💕 PIGGY BOOK server (API Server)
~~~js
cd server
npm install
npm run dev
~~~

### 💕 PIGGY BOOK client
~~~js
cd client
npm install
npm start
~~~
